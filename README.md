# Survival Game - OpenGL 🌲
A forest survival game where the player must eliminate aliens to level up.

## Technologies Used
- Language: C++
- Graphics Engine: OpenGL
- 3D Modeling: Blender (some imported models)

## Game Objective 🎯
- Eliminate a specific number of aliens to level up.
- Explore the forest while facing the aliens.

## My Contribution 
- Developed the game mechanics and enemy mobility using C++.
- Created sound effects and synchronized them with the game mechanics to achieve better immersion.
- Made major contributions to the creative development of the game.
- Creating animations in Blender for some animals. 
- Loading and rendering the animated models in OpenGL.
- Integrating animations into the game to ensure the characters move correctly.
